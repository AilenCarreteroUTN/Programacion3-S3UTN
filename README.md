## **Programación III - 2025**

Este repositorio contiene el desarrollo de los trabajos prácticos y parciales correspondientes a la cátedra Programación III a cargo del profesor **Gerardo Magni** de la Tecnicatura Superior en Programación de la Universidad Tecnológica Nacional, Facultad Regional Mendoza.

El enfoque de esta materia se expande hacia el desarrollo de aplicaciones web, la construcción de APIs, y la persistencia de datos, integrando tecnologías tanto de Backend (Java avanzado) como de Frontend (HTML, CSS, TypeScript). Además, se incorporan nuevos paradigmas como la programación funcional.

---

### 🚀 **Tecnologías y habilidades adquiridas.**
A través de los trabajos prácticos, se han consolidado las siguientes habilidades técnicas:
* **Desarrollo Web Frontend:** Estructuración semántica y accesible con HTML5, diseño y maquetación utilizando CSS avanzado (Modelo de caja, Flexbox, Variables, y metodología BEM).
* **Persistencia y Bases de Datos (JPA/ORM):** Diseño de diagramas de clases y su transformación al modelo relacional mediante Mapeo Objeto-Relacional (ORM), junto con la ejecución de consultas complejas a la base de datos utilizando JPQL.
* **Construcción de APIs y Patrones de Diseño:** Creación y consumo de servicios API REST, implementación del patrón DTO para desacoplar las capas de la aplicación y uso de la librería Lombok para optimizar y reducir el código repetitivo.
* **Programación Funcional en Java:** Implementación de expresiones Lambda, la API de Streams (filter, map, sorted, forEach) y el manejo de valores nulos mediante Optional.
* **Ecosistema JavaScript/TypeScript:** Tipado estricto y POO aplicada en el frontend con TypeScript, y creación de servicios y operaciones CRUD (Create, Read, Update, Delete) apoyándose en el entorno de Node.js.
* **POO Avanzada y Colaboración:** Resolución de problemas mediante la colaboración y relaciones bidireccionales entre objetos, diseño de arreglos dinámicos y algoritmos recursivos.
