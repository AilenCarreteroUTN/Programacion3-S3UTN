## **Práctico n°1A:** Interacció entre objetos.

Esta primera parte del trabajo práctico n°1 profundiza en la Interacción entre Objetos y el diseño de sistemas complejos, poniendo especial énfasis en la responsabilidad de cada clase para procesar datos y en la implementación de algoritmos recursivos sobre estructuras de asociación.

---

### 🛠️ **Tecnologías y conceptos cubiertos.**

* **Asociación y Navegabilidad:** Implementación de relaciones 1:N y N:N entre entidades para modelar procesos administrativos y civiles.

* **Recursividad en Estructuras de Datos:** Desarrollo de métodos recursivos para recorrer jerarquías de objetos, como listas de expedientes asociados o subsectores dentro de un sector.

* **Encapsulamiento de Lógica de Negocio:** Creación de métodos específicos dentro de las clases para realizar cálculos automáticos y concatenaciones de datos (carátulas, promedios, superficies).

* **Manejo de Excepciones de Negocio:** Implementación de validaciones críticas que disparan excepciones personalizadas ante datos inconsistentes (ej. superficie cubierta mayor a la del terreno).
